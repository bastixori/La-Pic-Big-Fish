# La Picá Big Fish 🐙

Landing page premium y responsiva para el restaurant **La Picá Big Fish** ubicado en Barrio República, Santiago, Chile.

## Características
- **Diseño Premium**: Interfaz moderna con temática marina oscura (Dark Marine) y detalles en turquesa y ámbar.
- **Glassmorphism**: Componentes visuales semi-transparentes con desenfoque de fondo y bordes iluminados.
- **Interactividad**: Menú interactivo por pestañas (Ceviches, Empanadas, Platos y Mariscos) y sistema dinámico de burbujas flotantes en el fondo.
- **Recursos Reales**: Integración del logotipo oficial de Instagram y fotos reales de los platos (como el pizarrón de empanadas del local).
- **Contacto**: Enlaces rápidos para pedidos por WhatsApp, Instagram y ubicación en Google Maps.

## Tecnologías Utilizadas
- HTML5 (Estructura semántica)
- CSS3 (Vanilla CSS con variables y animaciones personalizadas)
- JavaScript (Efectos de burbujas, menú dinámico e IntersectionObserver para animaciones al hacer scroll)
- Font Awesome (Iconografía)
- Google Fonts (Tipografías *Playfair Display*, *Inter* y *Bebas Neue*)
